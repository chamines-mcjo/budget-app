## Common Commands

The following are based on yarn which is our recommendation to work with this project.

#### Start project compiler
```
yarn start
```

#### Start for Android Simulator
```
yarn android
```

#### Start for iOS Simulator
```
yarn ios
```

#### Run all tests
```
yarn test
```

#### Run tests in watch mode
```
yarn test:watch
```
